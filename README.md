# Language-Translator
Language Translator is a responsive web-based application that allows users to translate text between multiple languages seamlessly. The application supports auto language detection, manual language selection, and provides a clean, modern user interface with dark and light mode support.

This project is built using HTML, CSS, and JavaScript, focusing on simplicity, usability, and performance. It is ideal for beginners who want to understand API integration, DOM manipulation, and UI design in JavaScript-based web applications.
